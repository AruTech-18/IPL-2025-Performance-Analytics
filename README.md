# 📊 IPL 2025 Performance Analytics

## 🔹 Overview
This project analyzes the data from the Indian Premier League (IPL) 2025 season using Python. The objective is to extract meaningful insights, identify performance trends, and visualize patterns influencing match outcomes.

---

## 🔹 Objectives
- Chasing vs. Defending: Analyzed winning strategies to identify dominance in the T20 format.
- Venue Intelligence: Calculated average 1st innings scores and match density per stadium to identify high-scoring grounds.
- Competitiveness Mapping: Used distribution plots to analyze winning margins.
- Team Consistency: Visualized total wins and performance trends across the tournament.

---

## 🔹 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔹 Key Analysis  
- Winning margin distribution  
- Team performance comparison  
- Distribution of matches across venues highlights the most frequently used stadiums in the IPL 2025 season.   

---

## 🔹 Key Insights
- Dominant teams identified based on wins  
- Winning margins reveal match competitiveness
